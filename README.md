# 🎓 AI College Enquiry Chatbot

## 📌 Project Overview

The **AI College Enquiry Chatbot** is a web-based application developed to assist students and visitors by providing instant information about the college. The chatbot answers common queries related to admissions, courses, departments, placements, hostel facilities, campus life, examinations, and contact information through an interactive chat interface.

This project aims to reduce manual effort in handling repetitive enquiries while improving accessibility to college information.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/25a35a0537/college_chatbot.git
```

### Navigate to the project folder

```bash
cd college_chatbot
```

### Create a virtual environment (Optional)

```bash
python -m venv venv
```

### Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
